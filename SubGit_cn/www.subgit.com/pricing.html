  • Commiters Limit Explained

    Registration key you've obtained may set limit on a number of committers. SubGit takes into account number of distinct committers in the Git history, that, upon initial Subversion to Git translation is completed, will also include all committers from the Subversion project history.

    In case number of distinct committers exceeds the limit, SubGit will start to issue friendly reminders about the necessity to upgrade your registration key; otherwise SubGit mirror will remain fully functional.
