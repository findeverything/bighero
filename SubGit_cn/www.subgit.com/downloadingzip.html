You may use SubGit for evaluation purposes without a registration as long as you like. During that evaluation period SubGit will remind you on a necessity of registration with the help of post commit and post receive messages.

By using SubGit you agree with a SubGit Licensing Agreement.

New in version 2.0.3 (complete release notes):

  • — Better handling of .gitignore file renames.
  • — Improved support for skip option.
  • — SSL client certificate could be specified on the command line of the import command.
  • — New version of SVNKit is used that improves SSL connectivity and GNOME keyring support.

System Requirements:

  • — Java 1.6 or newer version.
  • — We test SubGit on Linux, Windows and OS X systems.
